# TL;DR

## Problem Formulation

https://www.seas.ucla.edu/~vandenbe/publications/coneprog.pdf


Consider the cone quadratic program (cone QP) defined by 

$$
\begin{equation}
\begin{aligned}
  \text{minimize } 0.5x^TPx + c^Tx\\
  \text{subject to } Gx + s = h, Ax = b, \text{ and } s \succeq 0 
\end{aligned}
\end{equation}
$$
where $$P \sim n \times n, G \sim m \times n, A \sim p \times n, c \sim n \times 1, h \sim m \times 1, b \sim p \times 1$$. Note that the matrix $$P$$ must be positive semidefinite and the inequalities $$s \geq 0$$ is w.r.t a self-dual convex cone $$C$$, where $$C$$ is a cartesian product 

$$
\begin{equation}
C = C_1 \times \cdots \times C_K
\end{equation}
$$

where in this blog post and our Jax implementation, each $$C = C_1$$ is a nonnegative orthant. A nonnegative orthant is the set of points with nonnegative components ie. 

$$
\begin{equation}
\mathbb{R}_+^n =\{x \in \mathbb{R}^n \ | \ x_i \geq 0 \ \forall \ i\}
\end{equation}
$$

Note that any nonnegative scalar times a vector in a nonnegative orthant, this resulting vector is also in the nonnegative orthant, and so it follows that the nonnegative orthant is a special case of a cone. Thus, the notation $$\mathbb{R}_+^p$$ is used for the cone of 
nonnegative $$p$$-vectors.

## Logarithmic Barrier Functions

Log barrier functions are commonly used in constrained quadratic programming problems with respect to handling inequality constraints. For the special cone that is the nonnegative orthant, we introduce the function $$\phi: \mathbb{R}_+^p \to \mathbb{R}$$

$$
\begin{equation}
\phi(u) = -\sum_{j=1}^p \log u_j
\end{equation}
$$

To get some intuition, consider the constraint $$s \succ 0$$. As $$s_i \to 0^+, - \log s_i \to \infty$$ thus penalizing any solution that gets close to violating the constraint $$s \succ 0$$. To later on apply Newton's formula, we require knowledge of the first and second derivative -- ie. the gradient and Hessian -- of this log barrier function. The gradient $$g(u): \mathbb{R}_+^p \to \mathbb{R}^p$$ omits the mapping 

$$
\begin{equation}
g(u) = \nabla \phi(u) = - \text{diag}(u)^{-1} \textbf{1}
\end{equation}
$$

Similarly, the Hessian is the mapping $$H(u): \mathbb{R}_+^p \to \mathbb{R}^{p \times p}$$

$$
\begin{align}
H(u) = \nabla^2 \phi(u) = \text{diag}(u)^{-2}
\end{align}
$$
